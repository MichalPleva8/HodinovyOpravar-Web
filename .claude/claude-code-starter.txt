## CRITICAL RULES

**1. Component Choice:**
- ✅ DEFAULT to Astro components (`.astro`) for everything static
- ❌ Only use React (`.tsx`) for genuine interactivity:
  - ContactForm (validation, submission)
  - ImageGallery (lightbox)
  - MobileMenu (toggle state)
- Use `client:load` for forms, `client:visible` for galleries, `client:idle` for menus

**2. Styling:**
- ✅ Use Tailwind utility classes for ALL styling
- ✅ Reference design tokens: `text-h2`, `bg-primary`, `text-neutral-600`
- ❌ NO separate CSS files (unless special effects like complex animations)
- ✅ Mobile-first: base styles for mobile, add `md:`, `lg:`, `xl:` for larger screens

**3. Language & Tone:**
- Language: Slovak (Slovenčina)
- Tone: Professional but friendly, formal (vykanie - "vy" not "ty")
- Example: "Kontaktujte nás" not "Kontaktuj nás"

**4. File Organization:**
- Images: `/public/images/`
- Data: `src/data/`
- Components by type (layout, sections, ui, interactive)
- Pages in `src/pages/` (file-based routing)

**5. TypeScript:**
- Use TypeScript for all data files
- Define interfaces for data structures
- Type component props
